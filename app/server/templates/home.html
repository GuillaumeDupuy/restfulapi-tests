Get started

    API Endpoint

    http://127.0.0.1:8000/student

The Ynov API provides programmatic access to read Paris Ynov Campus data. Retrieve a student, provide an oauth connection, retrieve a subject, filter them, etc.

To use this API, you do not need an API key. Please contact us at guillaume.dupuy@ynov.com if you have any problems.

Get students

To see all students you need to make a GET call to the following url :
http://127.0.0.1:8000/student/

post students


# Here is a curl example
curl \
-X POST 'http://127.0.0.1:8000/student/' \
-H 'accept: application/json' \
-H 'Content-Type: application/json' \
-d '{
    "fullname":"Charles Fisman",
    "email":"charles.fisman@ynov.com",
    "course_of_study":"Master AI & Big Data",
    "year":4
}'

To post students you need to make a POST call to the following url :
http://127.0.0.1:8000/student/ 



Result example :

Update students

To update students you need to make a PUT call to the following url :
http://127.0.0.1:8000/student/

QUERY PARAMETERS

Field Type Description
id String The id of the student you want to modify the information

Delete students

To delete students you need to make a DELETE call to the following url :
http://127.0.0.1:8000/student/

QUERY PARAMETERS

Field Type Description
id String The id of the student you want to delete

Errors

The Ynov API uses the following error codes:

Error Code Meaning
X000 Student doesn't exist.
X001 There was an error updating the student data..
X002 Student with id {number_id} doesn't exist
